# Team Sorting App 

โปรแกรมเรียงลำดับตัวเลข (Sorting Application) พัฒนาด้วยภาษา Java โดยใช้การออกแบบแบบ **Strategy Pattern**

## Features
- รับค่าตัวเลข (Input) จากผู้ใช้งานได้
- เลือก Algorithm ในการเรียงลำดับได้ (Bubble Sort / Quick Sort)
- โครงสร้างโค้ดแยกส่วนกันทำงาน (Interface-based Design) ลดปัญหา Code Conflict

## Installation & Run
1. Clone โปรเจกต์ลงเครื่อง
2. เปิดโฟลเดอร์ `Project` ด้วย Eclipse หรือ IDE ที่ถนัด
3. รันไฟล์ `MainApp/Main.java`
